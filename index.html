HTML (HyperText Markup Language)

HTML is the foundational language used to create the structure and content of a web page. It uses tags to define elements like headings, paragraphs, links, images, and more.

CSS (Cascading Style Sheets)

CSS is used to control the appearance and layout of HTML elements on a web page. It allows you to apply styles like colors, fonts, spacing, and responsiveness, making the web page visually appealing

JavaScript

JavaScript is a programming language that adds interactivity and dynamic behavior to web pages. It enables you to create features like animations, form validations, and interactive games by manipulating HTML and CSS.
